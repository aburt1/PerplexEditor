AI Text Editor - Test Page

How to Test the Extension:

  1. Make sure the AI Text Editor extension is loaded in Firefox (about:debugging)
  2. Configure your Perplexity API key in the extension options
  3. Select text in any of the editable areas below
  4. Right-click and choose from the context menu options
  5. Watch the text transform using AI!

Troubleshooting

If the extension isn't working:

  • Check that the extension is loaded in about:debugging
  • Verify your Perplexity API key is set in the extension options
  • Open the browser console (F12) to see any error messages
  • Make sure you're selecting text in an editable area
  • Try refreshing the page and reloading the extension
